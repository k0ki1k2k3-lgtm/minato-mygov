🔒 このアプリは、家族構成・収入・障害の有無などの個人情報を
サーバーに送信・保存しません。すべての判定はあなたの端末内で行われます。

運営者：こっつん
連絡先：k.0k.i1k.2k.3@gmail.com

2. 収集する情報

本アプリが収集・取得する情報は以下に限られます。

  • Web Push通知の購読情報（endpoint）：通知を許可した場合のみ取得。ブラウザが発行する識別子で、氏名・住所・メールアドレス等は含みません。
  • ライフステージ情報：通知の配信タイミング制御のため、大まかなライフステージ（例：子育て中、シニアなど）をサーバーに保存する場合があります。

プロフィール（年齢・家族構成・収入・障害・就労状況など）はすべてあなたの端末内のみに保存され、外部サーバーには送信されません。

3. 収集しない情報

  • 氏名・住所・電話番号・メールアドレス
  • マイナンバー・各種証明書の内容
  • 正確な収入額・資産情報
  • 位置情報（GPSなど）

4. 情報の利用目的

  • Web Push通知の配信（制度の新着・申請期限のお知らせ）
  • 配信タイミングの最適化

取得した情報を広告目的で利用したり、第三者に販売・提供することは一切ありません。

5. 第三者提供

収集した情報を第三者に提供することはありません。ただし、以下のサービスを利用しています。

  • Cloudflare Workers / KV：Push通知の配信に使用（Push endpointを保存）
  • Google Analytics 4：アプリの利用状況の集計（ページビュー・操作イベント）。個人を特定する情報は収集しません。
  • GitHub Pages：アプリ本体・制度データベースの配信

6. データの削除

アプリ内「設定」タブの「すべてのデータをリセット」ボタンから、端末内のすべてのプロフィールデータを削除できます。

Push通知の購読を解除することで、サーバー側のendpoint情報も次回の配信サイクルで自動削除されます。

7. Cookieおよびローカルストレージ

本アプリはCookieを使用しません。プロフィールや利用状況の保存にブラウザのLocalStorageを使用します。これはあなたの端末内にのみ保存され、外部に送信されません。

8. お子様の利用

本アプリは特定の年齢制限を設けていませんが、プロフィールの入力には保護者の方のご確認をお願いします。

9. ポリシーの変更

本ポリシーを変更する場合は、このページの「最終更新」日付を更新します。重要な変更の際はアプリ内でお知らせします。
